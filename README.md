 🔐 Password Strength Analyzer

A simple and secure Password Strength Analyzer built using Python and HTML.  
This project evaluates password strength based on length, complexity, uniqueness, and entropy while also generating strong password suggestions.

✅ Check password length and complexity  
✅ Detect weak/common passwords  
✅ Prevent password reuse using SQLite database  
✅ Generate strong password suggestions  
✅ Calculate password entropy  
✅ Color-based strength meter (Red / Yellow / Green)  
✅ Secure password hashing using SHA-256 + Salt  
✅ Simple frontend using HTML, CSS, and JavaScript  

 🛠 Technologies Used

- Python
- HTML
- CSS
- JavaScript
- SQLite3
- SHA-256 Hashing
